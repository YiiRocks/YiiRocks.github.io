simple-html-dom configures voku/simple_html_dom.

Quick usage:
use YiiRocks\SimpleHtmlDom\SimpleHtmlDomInterface;

class Class {
    private SimpleHtmlDomInterface $dom;

    __construct(SimpleHtmlDomInterface $dom) {
            $this->dom = $dom;
    }
}
